## Marketing Website

### Prerequisites
- PHP 7.4 or greater
- Composer
- NodeJS
- MySQL

### How to deploy locally
- Clone repository locally.
- Run ```composer install```
- Run ```npm install && npm run dev```
- To start server run ```php artisan serve```
